# TradeVault — دستیار حرفه‌ای معاملات

یک پلتفرم مدرن و مینیمال برای ثبت معاملات، تحلیل عملکرد، مدیریت چالش‌های پراپ فرم و تصمیم‌گیری بهتر در معاملات.

## امکانات

### ژورنال اصلی
- **ثبت معاملات** با ورود/خروج/ضرر-سود/حد سود، جهت، وضعیت، محاسبه خودکار سود و زیان
- **پشتیبانی از چند حساب** — معاملات، فرصت‌ها و واچ‌لیست جداگانه برای هر حساب
- **قفل معاملات** — معاملات بعد از ۴۸ ساعت فقط‌خواندنی می‌شوند
- **پیگیری سود جزئی** — TP1/TP2/TP3 با چک‌باکس فعال‌سازی
- **تحلیل چند تایم‌فریم** — فیلدهای ۴ ساعته/۱ ساعته/۱۵ دقیقه برای هر معامله
- **یادداشت‌های مارک‌داون** — bold، italic، bullet در یادداشت‌ها و درس‌ها
- **پشتیبانی از اسکرین‌شات** — مستقیم paste کنید یا URL وارد کنید (URL‌های TradingView خودکار تبدیل می‌شوند)
- **تگ و احساسات** — وضعیت روانی خود را در هر معامله ثبت کنید
- **فیلدهای سفارشی** — فیلدهای دلخواه خود را تعریف کنید

### تحلیل‌ها
- **نرخ برد، ضریب سود، انتظاری، شاخص شارپ**
- **منحنی سرمایه** با گرادیان
- **نمودار دراوداون**
- **تقویم سود و زیان روزانه** با روزهای رنگی
- **تحلیل نمادها** — تعداد معاملات، نرخ برد، سود، بهترین/بدترین
- **تحلیل استراتژی** — عملکرد هر استراتژی
- **تحلیل زمانی** — سود بر اساس روز هفته و ساعت روز
- **همبستگی نمادها** — تشخیص پوزیشن‌های همپوشان
- **استریک‌ها** — بیشترین سری برد/باخت، استریک فعلی، ضریب بازیابی

### یکپارچه با چارت
- **چارت پیشرفته TradingView** با اندیکاتور RSI
- **واچ‌لیست سفارشی** همگام با نوار کناری TradingView
- **باقی ماندن نماد** — بازگشت به آخرین نماد مشاهده شده بعد از رفرش
- **تعویض سریع نماد** با mapping کامل (فارکس، کریپتو، فلزات، شاخص‌ها)

### صفحه فرصت‌ها
- **صفحه کامل** با فرم + لیست قابل جستجو و فیلتر
- **تصویر کوچک** با تبدیل خودکار URL‌های TradingView
- **اسلایدشو لایت‌باکس** — کلیک روی هر تصویر برای باز کردن تمام‌صفحه با ناوبری فلش
- **متن یادداشت** زیر هر تصویر در لایت‌باکس
- **کنترل کیبورد** — ← → برای ناوبری، Esc برای بستن

### تقویم اقتصادی
- **دانلود + آپلود** — دور زدن CORS با دانلود CSV از Forex Factory
- **چیدمان گروه‌بندی شده** — رویدادها بر اساس تاریخ با هایلایت "امروز"
- **نشانگر اهمیت** — نقاط رنگی و حاشیه رنگی (قرمز/زرد/خاکستری)
- **پرچم کشورها** — پرچم ایموجی برای ۵۰+ کشور
- **تبدیل منطقه زمانی** — EST، UTC، یا زمان محلی شما
- **جستجو و فیلتر** — بر اساس سطح اهمیت و نام رویداد

### محاسبه‌گر ریسک
- **فارکس/فلزات** — اندازه پوزیشن از درصد ریسک، ورود، ضرر‌سود، موجودی
- **کریپتو** — اندازه پوزیشن از مبلغ ریسک یا اندازه پوزیشن با درصد SL
- **محاسبه کارمزد** — کارمزد maker/taker برای کریپتو، دلار بر لات برای فارکس
- **سود خالص** — سود ناخالص منهای کارمزد

### ردیاب پراپ فرم
- **انواع چالش** ۱ مرحله‌ای، ۲ مرحله‌ای و فاندد
- **تشخیص خودکار مرحله ۲** — رسیدن به هدف مرحله ۱ → تغییر خودکار به مرحله ۲
- **سود خودکار** از معاملات مرتبط (بدون ورود دستی)
- **پیگیری ضرر روزانه** در مقابل حداکثر ضرر روزانه
- **نوارهای پیشرفت** برای هدف سود، ضرر روزانه و ضرر کل
- **اتصال معاملات** — انتخاب پراپ فرم در منوی کشویی فرم معامله
- **پاپ‌آپ تحلیل کامل** — کلیک روی هر کارت برای نرخ برد، ضریب سود، انتظاری، بهترین/بدترین معامله، تحلیل نماد و تاریخچه معاملات

### داشبورد ریسک
- ریسک امروز، این هفته، میانگین هر معامله، کل ریسک
- حداکثر دراوداون
- شاخص شارپ (سالانه شده)
- جدول ریسک هر معامله (۲۰ معامله اخیر)

### صفحه تنظیمات
- **تم پیش‌فرض** — روشن/تاریک
- **زمان سشن‌ها** — شخصی‌سازی ساعات آسیا/لندن/نیویورک
- **حساب‌ها** — افزودن/حذف/تعویض حساب
- **تنظیمات کارمزد** — درصد maker/taker کریپتو، دلار بر لات فارکس
- **مدیریت داده** — بک‌آپ، بازیابی، بارگذاری تقویم اقتصادی

## دو نسخه

### نسخه محلی (`index.html`)
- یک فایل HTML، بدون بک‌اند
- تمام داده‌ها در `localStorage` ذخیره می‌شوند
- مستقیماً در هر مرورگری باز کنید
- خروجی/ورودی بک‌آپ به صورت JSON

### نسخه Cloudflare Worker (`worker.js`)
- Cloudflare Worker تک‌فایل
- دیتابیس D1 برای ذخیره‌سازی دائمی
- KV برای مدیریت سشن
- احراز هویت با رمز عبور
- همان رابط کاربری، با همگام‌سازی API برای تمام داده‌ها

## نصب

### نسخه محلی
1. فایل `index.html` را دانلود کنید
2. در هر مرورگر مدرنی باز کنید
3. شروع به ثبت معاملات کنید

### نسخه Worker
1. به Cloudflare Dashboard → Workers & Pages → Create Worker بروید
2. کل فایل `worker.js` را به عنوان اسکریپت Worker کپی کنید
3. به Settings → Variables بروید: `PASSWORD = "رمز-عبور-شما"` را اضافه کنید
4. به Settings → D1 database bindings بروید: یک دیتابیس D1 بسازید، به عنوان `DB` متصل کنید
5. به Settings → KV namespace bindings بروید: یک KV namespace بسازید، به عنوان `SESSIONS` متصل کنید
6. ذخیره و دیپلوی کنید
7. اسکیمای دیتابیس به‌طور خودکار در اولین درخواست ایجاد می‌شود

## میانبرهای کیبورد
- `←` / `→` — ناوبری اسلایدشوی تصاویر فرصت‌ها
- `Esc` — بستن لایت‌باکس / مودال‌ها

## داده و حریم خصوصی
- نسخه محلی: تمام داده‌ها در localStorage مرورگر شما می‌مانند
- نسخه Worker: داده‌ها در Cloudflare D1 ذخیره می‌شوند، هرگز با اشخاص ثالث به اشتراک گذاشته نمی‌شوند
- بدون تحلیل، بدون ردیابی، بدون تبلیغات
- بک‌آپ/بازیابی داده‌های خود را هر زمان به صورت JSON
